Dispatching and Repositioning System

This project is Aimed at optimizing the efficiency of an online ride-hailing application (Didi) by shortening users' waiting time and drivers' pick-up time, to improve the cumulative income of the platform. We built a grid-based simulator, to score different models by the adjusted rewards within a time period. The model is based on the Mean-Field Deep Q Learning algorithm, designed a more stable and efficient Reinforcement Learning algorithm, which is capable of handling large scale input data, to meet dynamic traffic situations, and tuned parameters for this model by experiments.

Humor Generator

Aimed at building a deep model that is able to generate humors automatically. Clawed short humors (Conan talk show sentences) from teamcoco and wocka, and cleaned them. For each short humor, set its pro-nouns set as its feature. Applied RNN in Encoder and Decoder, and introduced Attention mechanism among them. Fed in features as input to the encoder, and aimed at obtaining the original humors word by word. Prediction is finished by providing random combinations of pro-nouns. The meaningful rate is around 11%.

Friend Recommendation System

Aimed at building a friend-matching system based on personal musical tastes.For a fixed group of users, extracted their favorite songs. For each song, collected all of its comments, and transferred high-frequency words into tf-idf maxtrix as its feature. For each user, viewed his/her feature as an average of all related song features. Considered the Manhattan distance among users as the user difference distances. A small difference distance of two users indicates a high similarity between them.
