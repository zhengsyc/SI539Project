Dispatching and Repositioning System on a Ride-hailing Application (Didi)
  • Objective: Aimed at optimizing the efficiency of an online ride-hailing application (Didi) by shortening users' waiting time and drivers' pick-up time, to improve the cumulative income of the platform.
  • Environment Setup: Built a grid-based simulator, to score different models by the adjusted rewards within a time period.
  • Model Improvements: Based on the Mean-Field Deep Q Learning algorithm, designed a more stable and efficient Reinforcement Learning algorithm, which is capable of handling large scale input data, to meet dynamic traffic situations, and tuned parameters for this model by experiments.
  • Performance: Beated the best baseline (Markov Decision Process Model) by 7% after the model converged.

Neural Humor Generator Using RNN Model
  • Objective: Aimed at building a deep model that is able to generate humors automatically.
  • Dataset Preparation: Clawed short humors (Conan talk show sentences) from teamcoco and wocka, and cleaned them. For each short humor, set its pro-nouns set as its feature.
  • Model Setup: Applied RNN in Encoder and Decoder, and introduced Attention mechanism among them.
  • Training and Prediction Mechanism: Fed in features as input to the encoder, and aimed at obtaining the original humors word by word. Prediction is finished by providing random combinations of pro-nouns. The meaningful rate is around 11%.
